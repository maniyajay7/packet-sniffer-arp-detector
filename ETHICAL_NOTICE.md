# ⚖️ Ethical and Legal Notice

## Authorised Use Only

This tool — **Advanced Packet Sniffer + ARP Spoofing Detector** — is designed
for **defensive network monitoring** and **educational purposes only**.

By using this software, you agree to the following:

### ✅ Permitted Uses

- Monitoring traffic on **your own network** (home lab, personal devices).
- Analysing traffic on networks where you have **explicit, written
  permission** from the network owner/administrator.
- Using sample/simulated traffic (PCAP files) in a **controlled lab
  environment** for learning and experimentation.
- Academic projects, coursework, and cybersecurity practical submissions
  where safe, ethical testing is documented.

### ❌ Prohibited Uses

- Sniffing traffic on **public networks** (coffee shops, airports, etc.)
  without authorisation.
- Monitoring traffic on **corporate or institutional networks** without
  formal, documented approval.
- Using this tool as a component of any **offensive attack**, including but
  not limited to ARP spoofing/poisoning, man-in-the-middle attacks, or
  credential interception.
- Any activity that violates local, national, or international **computer
  crime laws**.

### 🛡️ No Offensive Code

This project **does not contain** any code to perform ARP spoofing, ARP
poisoning, or any other network attack.  The detection logic is strictly
**passive** — it only reads and analyses packets; it never injects, modifies,
or retransmits traffic.
